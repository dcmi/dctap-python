dctap
=====

dctap is a module and command-line utility for reading, interpreting, and verifying CSV files formatted according to the `DC Tabular Application Profiles (DCTAP) model <https://github.com/dcmi/dctap/blob/main/TAPprimer.md>`_. Development of the module is taking place `on Github <https://github.com/dcmi/dctap-python>`_.
