dctap
=====

**dctap** is a module and command-line utility for reading and interpreting CSV files formatted according to the `DC Tabular Application Profiles (DCTAP) model <https://github.com/dcmi/dctap/blob/main/TAPprimer.md>`_. This module is being developed on Github at `https://github.com/dcmi/dctap-python <https://github.com/dcmi/dctap-python>`_.
