dctap-python

Basic validation of a DCTAP instance.

Installation
------------

.. code-block:: bash

   $ git clone https://github.com/dcmi/dctap-python.git
   $ cd dctap
   $ python -m venv .venv
   $ source .venv/bin/activate
   $ pip install flit
   $ flit install -s
