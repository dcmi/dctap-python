**dctap** subcommands
^^^^^^^^^^^^^^^^^^^^^

The utility has two subcommands:

- **dctap init** generates a customizable configuration file.
- **dctap read** reads a CSV file, performs consistency checks, and outputs normalized views of the CSV contents to text (by default to the screen), JSON, or YAML.
