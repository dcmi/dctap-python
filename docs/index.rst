dctap
=====

dctap is a command-line utility for reading, interpreting, and verifying CSV files formatted according to the DC Tabular Application Profiles (DCTAP) model. 

This project has been undertaken in parallel to, and in support of, a DCMI working group which is creating the DCTAP model [#dcapig]_.
